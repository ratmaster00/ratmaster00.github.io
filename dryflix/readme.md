# 🎬 DryFlix — Watch Paint Dry

> *The world's most important streaming experience.*

A fully responsive, mobile-first fake streaming website dedicated to the singular cinematic masterpiece: **Watch Paint Dry** (10 hours, uncut). Built with vanilla HTML, CSS, and a tiny sprinkle of JavaScript. No frameworks. No dependencies. No paint.

![Black and purple streaming UI with a play card, card rows, and reviews](https://img.shields.io/badge/status-streaming-blueviolet?style=flat-square) ![License: MIT](https://img.shields.io/badge/license-MIT-purple?style=flat-square) ![No frameworks](https://img.shields.io/badge/dependencies-none-black?style=flat-square)

---

## ✨ Features

- **Streaming UI** — looks like a real video platform (it is not)
- **Animated play card** — paint drips, glowing play button, links out to YouTube
- **Session timer** — tracks how long you've been watching; upgrades your *patience level* from "Just started" all the way to "One with the paint"
- **Horizontal scroll rows** — "More Like This" and "Trending in Slow Cinema" with cards for similarly important titles (Grass Growing Live, Rock: A Biography, etc.)
- **Film details grid** — Director: The Weather. Audio: Dolby Silence™
- **Fake reviews** — five stars across the board
- **Mobile bottom nav** — Home, Search, My List, Profile (all decorative)
- **Fully responsive** — works on phones, tablets, and the desktop monitor you definitely haven't been staring at all day
- **Zero dependencies** — single `.html` file, ships with two Google Fonts

---

## 🚀 Usage

Just open the file. Seriously.

```bash
git clone https://github.com/yourname/dryflix.git
cd dryflix
open watch-paint-dry.html   # macOS
xdg-open watch-paint-dry.html  # Linux
```

Or drag it into any browser. That's it. There is no build step. There is no server. There is only paint.
